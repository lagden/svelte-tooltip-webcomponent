Svelte component

## Install

```
$ npm i -S @tadashi/tooltip-webcomponent
```


## Usage

```html
<script
	type="module"
	src="./node_modules/@tadashi/tooltip-webcomponent/dist/Tooltip.js"
></script>

<tadashi-tooltip right="true" size="200">Ajuda aqui</tadashi-tooltip>
```
